# Documentação do Projeto

Este diretório contém a documentação oficial (PT-BR) do repositório.

## Índice

1. `how-to/build-cmake.md`

   Como compilar o firmware usando CMake e os *presets* do projeto.

2. `how-to/debug-openocd.md`

   Como gravar e depurar usando OpenOCD + GDB.

3. `how-to/debug-stutil.md`

   Como depurar usando `st-util` + GDB (alternativa ao OpenOCD).

4. `how-to/troubleshooting.md`

   Problemas comuns e soluções.

5. `architecture.md`

   Visão de arquitetura: como o firmware está organizado (CubeMX/HAL + driver do e-Paper).

6. `REQUIREMENTS.md`

   Requisitos e premissas (hardware/software).

7. `PLAN.md`

   Plano/roadmap (itens de melhoria sugeridos).

8. `CHANGELOG.md`

   Histórico de mudanças.
